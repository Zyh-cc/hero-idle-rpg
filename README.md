# ⚔️ 勇者闯关挂机物语

> 一款纯网页单文件像素风挂机 RPG，无需安装，打开即玩！

**[🎮 立即游玩 →](https://zyh-cc.github.io/勇者闯关挂机网页小游戏/)**

---

## 🎯 游戏简介

你是一位身怀绝技的勇者，踏入充满危机的魔王领地。
通过挂机战斗不断积累金币、升级技能、购买装备，
挑战越来越强大的精英怪与 BOSS，最终击败终极魔王！

- **挂机不离线** — 战斗自动推进，离开也能慢慢养成
- **策略深度** — 技能树、装备、召唤物、宠物多维搭配
- **转生系统** — 击败魔王后转生，携带加成再战更强关卡
- **像素战斗** — Canvas 实时渲染战斗动画，爽快直观

---

## ✨ 核心玩法

| 系统 | 说明 |
|------|------|
| 🗡️ 战斗系统 | 基于 Tick 的回合制战斗，暴击、连击、吸血、格挡 |
| 🌳 技能树 | 三大流派：召唤流 / 骑士流 / 辅助流，自由组合 |
| 🐲 召唤系统 | 召唤骷髅、死亡骑士、暗黑元素，打造召唤军团 |
| 🐾 宠物系统 | 捕捉宠物提供战斗加成，多种宠物各有专属技能 |
| 🏪 商店 | 购买装备、药水、特殊道具持续强化勇者 |
| 👑 BOSS 挑战 | 多阶段 BOSS 拥有联动技能链，考验策略与时机 |
| 🔄 转生系统 | 转生保留加成，难度与奖励同步提升 |

---

## 🎮 操作说明

- **无需操作** — 战斗全自动，专注策略即可
- **战斗速度** — 右上角可调节 ×1 ~ ×5 速度倍率
- **技能释放** — 部分主动技能点击手动触发
- **存档** — 游戏进度自动保存至浏览器本地

---

## 👾 敌人阵容

- **普通怪物** — 随关卡强度动态缩放，永远有挑战
- **精英怪** — 拥有特殊技能（瘟疫、召唤、石化…）
- **小 BOSS** — 多阶段战斗，阶段切换触发连锁技能组
- **终极魔王** — 恶魔领主 · 骸骨暴君 · 远古巨龙三大终极挑战
